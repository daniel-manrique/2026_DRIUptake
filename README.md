# 2026 DRI Uptake: From Friction to Sovereignty

This repository contains a Quarto-based Reveal.js presentation developed for the Research Data Alliance (RDA) plenary/presentation. It explores the concept of "uptake" within the Canadian Digital Research Infrastructure (DRI) ecosystem, moving beyond mere usage metrics towards seamless integration, reduced friction, and data sovereignty.

## How to View Locally

1. Install [Quarto](https://quarto.org/docs/get-started/).
2. Clone this repository.
3. Run the following command in your terminal:
   ```bash
   quarto preview rda_canada_uptake.qmd
   ```
